# Database 
- Mongodb: npm install mongoose
  + mongoose.connect(process.env.MONGO_URL)
- import axios from ' axios '
- axios version: 1.6.2
- set up MONGO_URL in backend .env
# Backend
- npm init --yes
- npm install express
- npm i dotenv cors react-router-dom

- ReactHotToast: npm install react-hot-toast
